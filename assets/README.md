# Assets

Place SANANKA photos in `assets/images/`.

Suggested filenames:

- `hero-portrait.jpg` for the main hero image
- `runway.jpg` for the modeling section
- `editorial.jpg` for the smaller modeling image
- `gallery-stage.jpg`
- `gallery-culture.jpg`
- `gallery-brand.jpg`
- `gallery-impact.jpg`
- `gallery-festival.jpg`

After adding images, replace the placeholder blocks in `index.html` with `<img>` tags or set them as CSS background images.
